05   Website Design — Orchids.ai Prompt
 
Orchids prompt — restyle existing site
Paste the following prompt directly into Orchids to restyle your existing site with the Hanfan brand identity. Adjust section descriptions to match your actual content.
 
Orchids Restyle Prompt — copy and paste:
Restyle my existing website with the following brand identity, keeping all current content and structure intact. Do not change any copy, headings, or page sections. Only update the visual design.

BRAND: Hanfan — on-demand home services marketplace. 'Real pros on demand.' / 'Have a need. Fill a need.'
COLORS: Primary coral #FF5A5F, secondary amber #FFB400, ink #1A1A1A, warm white #FFF8F4, ash #6B6B6B. Use coral for all primary CTAs, buttons, active states, and accents. Use amber for secondary badges and pro-side elements.
FONTS: Import from Google Fonts — Syne (weights 700, 800) for all headings and display text; DM Sans (weights 300, 400, 500) for all body copy and UI elements.
STYLE: Bold, warm, human — Airbnb energy. Rounded corners (border-radius 12–16px on cards, 100px on pills/chips, 10–12px on buttons). Clean white surfaces with subtle warm-tinted backgrounds (#FFF8F4). Generous whitespace. No gradients, no heavy shadows — flat and modern.
BUTTONS: Primary — coral background #FF5A5F, white text, DM Sans 500, 14–16px, border-radius 10px, padding 12px 24px. Secondary — transparent background, coral border, coral text.
NAVBAR: White background, Hanfan wordmark in Syne 800 #1A1A1A (no logo image needed), coral CTA button on the right.
HERO SECTION: Coral background #FF5A5F. Headline in Syne 800, white, large. Tagline 'Real pros on demand.' in DM Sans 400, white, slightly muted. Ghost a handshake illustration at 12% opacity white in the background right side.
DARK MODE: Dark background #1A1A1A, card surfaces #2A2A2A, coral accents preserved, white body text, ash for secondary text.

09   Developer Handoff Checklist
 
Before development
— Confirm hanfan.app domain is active and SSL enabled
— Import Syne and DM Sans from Google Fonts in the <head>
— Set up CSS custom properties from Section 02 in :root
— Configure dark mode with prefers-color-scheme media query
— Source the approved Hanfan logo from the brand identity image provided
 
Orchids.ai workflow
— Paste the Orchids prompt from Section 05 to restyle the site
— Review output — check all coral CTAs, font rendering, and card styles
— Export clean code from Orchids once approved
— Deploy to hanfan.app via Orchids hosting or export to Vercel/Netlify
 
Quality checklist
— All H1/H2/H3 use Syne font family
— All body copy and UI labels use DM Sans
— Primary buttons are coral #FF5A5F with white text
— Hero section has coral background with ghosted handshake watermark
— Card border-radius is 16px consistently
— No gradients, drop shadows, or heavy visual effects
— Dark mode tested on iOS and Android
— Logo shows 'Hanfan' in Syne 800 — not the old logo file
— Tagline 'Real pros on demand.' appears in hero
— Mobile viewport tested at 375px, 390px, and 428px widths
